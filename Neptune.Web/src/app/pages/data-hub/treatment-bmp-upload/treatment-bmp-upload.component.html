Use this form to bulk upload Treatment BMPs from a CSV. Requirements:

  • Select the Treatment BMP Type the rows in the CSV represent. All rows must share that type.
  • Required columns: Jurisdiction, BMP Name, Latitude, Longitude, Sizing Basis, Trash Capture Status, Owner.
  • Optional columns: Year Built or Installed, Asset ID in System of Record, Required Lifespan of Installation, Allowable End Date of Installation, Required Field Visits Per Year, Required Post-Storm Field Visits Per Year, Notes.
  • If a BMP with the same Name and Jurisdiction already exists, its record will be updated; otherwise a new record is created.
  • Custom attributes for the selected Treatment BMP Type can also be included as columns and will be upserted.
