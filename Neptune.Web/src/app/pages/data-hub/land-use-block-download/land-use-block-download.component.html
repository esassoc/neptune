Use this form to download Land Use Blocks for a given jurisdiction:

  • A single ArcGIS file geodatabase containing land use block features will be produced for the selected jurisdiction.
  • It will contain attributes for Block ID, Land Use Type, Jurisdiction, Block Area, Trash Generation Rate, Land Use Description, Median Household Income Residential, Median Household Income Retail, Trash Results Area, and Permit Type.
  • This file download is intended to support bulk editing in desktop GIS software and can be re-uploaded to the platform to replace the existing land use blocks utilized in trash results calculations.
