Build a single ArcGIS file geodatabase containing all Treatment BMPs you have permission to view, organized into one feature class per BMP Type.

  • Each Treatment BMP feature carries Name, Jurisdiction, Type, Owner, Year Built, System of Record ID, WQMP, Trash Capture Effectiveness, Notes, last assessment metadata, lifespan, and required field visit cadence.
  • Feature classes for each BMP Type also include the type's custom maintenance attributes.
  • The download is filtered to the jurisdictions you are assigned to. Administrators receive all BMPs.
  • Note: Generation may take a moment depending on the volume of inventory data. Thank you for your patience.
