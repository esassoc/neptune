Use this form to bulk upload On-land Visual Trash Assessments (OVTAs) from an XLSX file. Requirements:

  • The file must match the OVTA bulk upload template, with an "OVTA Assessments" worksheet.
  • Each row references an existing OVTA Area (by Area Name) and an existing user (by Created By Person email).
  • Required columns: Area Name, Jurisdiction Name, Created By Person, Status (Finalized or Draft), Score (A, B, C, D), Is Progress Assessment (Yes/No), Completed Date.
  • Preliminary Source Identification Type columns may include comma-separated lists. "Other" is not supported via bulk upload.
  • After upload, baseline and progress scores are recalculated automatically for the affected OVTA Areas.
