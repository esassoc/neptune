  • The feature class must include attributes: PriorityLandUseType, LandUseDescription, TrashGenerationRate, LandUseForTGR, MedianHouseholdIncomeResidential, MedianHouseholdIncomeRetail, PermitType.
  • Validated rows are written to the staging table; a background job then processes them and emails the result to you. You can close this page once the upload is staged.
