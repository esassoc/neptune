Use this form to bulk upload Water Quality Management Plans from an XLSX file. Requirements:

  • The file must match the WQMP bulk upload template, with a "WQMP" worksheet.
  • Required columns: WQMP Name, Land Use, Priority, Status, Development Type, Trash Capture Status.
  • Optional columns include Maintenance Contact details, Permit Term, Hydromodification Controls Apply, Approval Date, Date of Construction, Hydrologic Subarea, Record Number, Recorded WQMP Area (Acres), Trash Capture Effectiveness, Modeling Approach.
  • If a WQMP with the same name already exists in the selected jurisdiction, its record will be updated; otherwise a new record is created.
