Use this form to bulk upload Trash Screen Field Visit assessments and maintenance records from an XLSX file. Requirements:

  • The file must match the Trash Screen Field Visit template, including the "Field Visits" worksheet and the expected column headers.
  • Each row's BMP Name and Jurisdiction must match an existing Inlet and Trash Screen BMP that you have permission to manage.
  • Initial Assessment, Maintenance Record, and Post-Maintenance Assessment sections are each independently optional, but every field within a section must be filled out if any one of them is.
  • If a Field Visit already exists for the same BMP and date, its existing initial / post-maintenance assessment and maintenance record will be updated; otherwise new records are created.
  • Field Visit dates are interpreted as Pacific Standard Time and stored in UTC.
