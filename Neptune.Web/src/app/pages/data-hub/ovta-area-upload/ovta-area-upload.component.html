Download OVTA Assessment Areas

Use this form to bulk upload Onland Visual Trash Assessment Areas. Requirements:

  • A single ArcGIS file geodatabase containing only Onland Visual Trash Assessment Area features.
  • The file geodatabase (.gdb) must be contained within a compressed zip file (.zip).
  • The zip file must contain only one file geodatabase and no other data.
  • The file geodatabase must contain only one feature class.
  • You must supply the name of the attribute representing the area name. The default is OVTAAreaName.
  • Geodatabases using ESRI Smart Database Compression (SDC) are not supported. Please uncompress before zipping and uploading.
  • Version 10.1+ file geodatabases are supported.
  • Areas with names that already exist in the selected jurisdiction will have their geometry and description updated; new names are created.
  • After upload you'll be taken to a review page to confirm before any changes commit.
