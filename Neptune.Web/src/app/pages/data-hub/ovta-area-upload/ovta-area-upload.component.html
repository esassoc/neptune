Use this form to bulk upload OVTA assessment areas from a zipped File Geodatabase. Requirements:

  • A single ArcGIS file geodatabase containing only OVTA Area features (.gdb inside a .zip).
  • The feature class must contain a unique area-name attribute and an optional Description attribute.
  • The default area-name field is OVTAAreaName — supply the field name your GDB uses if different.
  • Areas with names that already exist in the selected jurisdiction will have their geometry and description updated; new names are created.
  • After upload you'll be taken to a review page to confirm before any changes commit.
